# PDF Question Answering Assistant Task

## Scenario:
You are working in a legal-tech company that handles large numbers of legal
agreements and policy documents daily.
The company wants to build an AI-powered assistant that allows users to upload PDF or
DOCX files and ask questions about the document in natural language.
The goal is to reduce the time required to review long documents and quickly retrieve
important information.

**Input**: PDF or Docx
**Output**: AI Generated Answers

## Notes:

While implementing the solution, consider:
 Object-Oriented Programming (OOP) principles
 Clean architecture and modular design
 Appropriate use of design patterns

## Deliverables:

 Source Code (Don’t forget to remove any API keys used)
 README
 Setup Instructions
 Sample queries and outputs

## Bonus Points:

 Streaming responses
 Conversation history
 Docker support
